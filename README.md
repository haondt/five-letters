# five-letters

A solution to Matt Parkers [5-word-problem](https://www.youtube.com/watch?v=_-AfhLQfb6w).

# Usage

get `words_alpha.txt` from https://github.com/dwyl/english-words, and run

```shell
python3 search.py
```
